=> 2번째 인덱스에서 충돌 발생( C와 A 불일치)


나쁜 문자 이동 테이블 사용시:
충돌이 발생한 본문의 문자: C
충돌이 발생한 패턴의 문자 인덱스: 2

C의 ascii 코드상 값 67, 나쁜 문자 이동 테이블[67] == 0

충돌이 발생한 패턴의 문자 인덱스 - 나쁜 문자 이동 테이블[67]
=> 패턴이 본문 기준으로 이동해야할 크기 == 2


착한 접미부 이동 테이블 사용시:
충돌이 발생한 패턴의 문자 인덱스: 2
충돌이 발생한 패턴의 문자 인덱스 뒤 접미부의 시작위치: 3

착한 접미부 이동 테이블[3]의 이동거리 == 2
=> 패턴이 본문 기준으로 이동해야할 크기 == 2


나쁜 문자 이동 테이블 사용시의 이동 크기와
착한 접미부 이동 테이블 사용시의 이동 크기가 깉으므로
패턴을 본문 기준으로 2만큼 이동

==> ADCDADACCB
      CDADA

패턴과 본문을 비교하는 본문의 시작위치가 인덱스 2일때 본문상에서 같은 패턴을 찾을 수 있습니다.
